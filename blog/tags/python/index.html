pyenv是一个python版本管理工具, 可以方便的管理多个python版本.
对于开发者来说有的时候会有多个python版本的需求.
例如, 有的项目需要python3.6, 有的项目需要python3.7. 这个时候就需要使用pyenv来管理多个python版本.

pyenv是一个轻量级的工具, 但是功能也很强大, 适合于开发者使用.
