# Gitlab中常见的标签

`Gitlab`中的标签是一个非常有用的功能, 该功能可以帮助我们更好的管理`ISSUE`.<br/>
下面介绍一下`Gitlab`中常见的标签的定义和其使用希望能够对你有所帮助.<br/>

<!--truncate-->

## 标签的定义

下面介绍常见的一些标签的定义, 以便我们更好的使用.

export const Label = ({name, color}) => (
	<span style={{
		fontSize: "0.75rem",
		color: "white", // 设置字体颜色为白色
		background: color,
		borderRadius: "22px", // 设置边框的圆角为5像素
		padding: "4px 12px", // 设置内边距为2像素上下，5像素左右
		margin: "2px", // 设置外边距为2像素
	}}>{name}</span>
)

:::note
Improvement和Enhancement之间的区别<br/>

Improvement: <br/>
表示对一个不太好的事物进行改善或提高, 使其达到更好的水平或状态.<br/>
We need to improve our product quality to meet the customer’s needs.(我们需要改善我们的产品质量, 以满足客户的需求.)<br/>

Enhancement: <br/>
表示对一个已经很好的事物进行增强或扩展, 使其更加优秀或有吸引力.<br/>
The new feature of this phone enhanced its competitiveness.(这款手机的新功能增强了它的竞争力.)<br/>
:::
