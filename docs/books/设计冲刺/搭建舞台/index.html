搭建舞台

识别挑战​

从大问题入手​

找出目前发展最重要的问题

挑战越大，设计冲刺的效果越好​

设计冲刺能够解决的三种挑战和困难​
  • 风险高 当面临着一个重大问题并且解决方案可能会耗费大量的时间或资金时, 设计冲刺能够帮助你在马力全开之前看清航海图。
  • 时间紧 当面临着严格的最后期限， 并且需要好而且快的解决方案。而设计冲刺，顾名思义，正是为速度而生。
  • 起步难 有些重要的项目很难启动。有些则在中途失去动力。设计冲刺就像一个火箭加速器，带来新方法和新视角，帮助你摆脱地心引力的束缚，顺利找到解决方案。
结论​
  • 选择最重要的问题
  • 开展设计冲刺需要大量的精力和专注力。不要着眼于小胜利，或是可有可无的项目
多大的问题才算大​

没有什么问题大到不能用设计冲刺来解决。

  • 设计冲刺逼着你的团队专注解决最紧迫的问题。在仅有的一周时间内，不可能一次性解决所有的问题，这样可以确保你们取得进展。
  • 设计冲刺让你可以通过一个成品的外观获得反馈。

先解决外观问题​

外观很重要。它是你的产品和服务给用户留下的第一印象。人类复杂多变，因此完全无法预料他们对一个全新的解决方案会有何反应。如果我们的新点子失败了，通常是因为我们太过高估了用户的理解力和需求点。 先做好外观，然后你可以倒回去解决背后的系统或技术问题。专注于外观能让你先快速取得进展，解决重大问题，然后再付诸实践，因此无论多大的挑战都能从设计冲刺中受益。

组建团队​

确定一名决策者、一名引导者以及多样化的队员​

决策者是一个有权威、能够做决定的人。决策者是这个项目正式负责决策的人。

确定一位或两位决策者​

决策者必须参与设计冲刺

说服决策者加入的理由​
  • 快速取得进展
  • 只是个试验
  • 解释你的取舍
  • 重点是专注
如果决策者没时间怎么办​
  • 邀请她在几个关键节点参与进来。
    • 周一，她可以分享她对待亟须解决的问题的看法。
    • 周三，她可以帮忙选出最佳方案以待测试。
    • 周五，她应该来看看用户对原型有什么评价和反应。
  • 正式指定团队中的一人做她的代表。

团队人数​

团队最理想的人数是七人或更少。

团队角色​

角色职责参考人选
决策者团队中做决定的人首席执行官，创始人，产品经理，设计总监
财务专家解释资金来源或流向首席执行官，首席财务官，商务拓展经理
营销专家确定你们公司对外传播的信息首席营销官，营销专家，公关经理，社区经理
客服专家经常与你们的用户进行一对一的谈话研究员，销售人员，客服人员
技术专家了解你们公司能够实现和提供的服务首席技术官，工程师
设计专家负责设计你们公司的产品设计师，产品经理

有请捣乱分子​

我们说的不是那些纯粹为辩论而辩论的人。我们说的是那些头脑聪明、持有强烈反对观点，而且有他参与你可能会感觉有点儿别扭的人。 如果捣乱分子参与讨论，哪怕只是偶尔出场，她会有参与感和被重视感。不过还有一个更重要的原因。捣乱分子与在场其他人看问题的角度完全不同。他们想到的奇特方案也许正中核心。

邀请多出来的专家参与星期一的讨论​

邀请多出来的专家星期一下午过来开个小会。在会上，他们可以与团队成员分享他们所了解的信息以及他们的观点。（再过几页，就是“请教专家”过程的详细介绍。）一位专家半个小时应该足够了。

选一个引导者​

她应该对组织会议有信心，既能总结讨论要点，又能在恰当的时机结束争论，进入下个环节。 引导者需要对各种决定保持中立。让同一个人担任决策者和引导者角色可不是个好主意。一般来说，让一个之前没有和团队成员合作过的人担任引导者，效果不错，不过这不是强制要求。

确定时间和地点​

定出五天连续工作日，找到合适的会议室​

  • 注意力碎片化对工作效率有负面影响

    在工作中受到干扰的人平均需要23分钟才能恢复受干扰前的工作状态。（在看完刚收到的短信之后，我们打算再看更多的研究成果。）

  • 有意义的工作需要一大段不受干扰的时间
  • 通过召集合适的团队成员，合理安排活动，并且减少干扰，在合理的时间内快速取得进展
  • 成员必须经历充沛, 注意力高度集中

在日程表上空出一整个星期​

团队必须确保星期一到星期四每天早上10点到下午5点都在一起。​
为什么定成五天​
  • 更短的时间搞得大家都精疲力竭
  • 更长的时间周末会影响工作的连续性。干扰和拖延会悄悄出现。
  • 五天时间带来的紧迫性让人们更容易集中注意力、打断不必要的争论，同时又有喘息的空间来制作原型、进行测试，不至于忙到吐血
  • 由于大部分公司都实行五天工作制，把设计冲刺安排到现有的日程规划中切实可行
时间安排​
  • 上午可以休息一次（11:30左右）
  • 一个小时的午饭时间（13:00左右）
  • 下午可以休息一次（15:30左右）

无电子设备原则​

在设计冲刺期间，时间非常宝贵​
电子设备会吸走冲刺的动力​
例外情况​
  • 在休息时间使用
  • 离开房间使用
  • 完成一些特别任务时, 例如: 需要向团队全体成员展示东西的时候，还有星期四制作原型的时候。

使用白板最大化利用时间和注意力​

  • 狭小的白板阻碍进度
  • 设计冲刺开始之前，一定要检查白板是否到位
  • 人类的短时记忆并不是很好，擅长空间记忆。一个贴满各种纸条、图表、打印资料的冲刺室，正好利用了空间记忆

准备两块大白板​

你至少需要准备两块大白板。这样才有足够的空间来进行大部分活动（随着过程进展，你还会需要拍照、擦除部分内容，或者重新组织内容分布位置），并且保持重要的信息一整个星期都可见。

当没有白板时的替代方法​

  • 旋转白板
  • IdeaPaint涂料
  • 白纸
