# Let it Crash(让其崩溃)

## 简介

`Let it Crash`
是一种软件开发方法，它认为软件应该尽可能地崩溃，而不是尽可能地避免崩溃。<br/>
这种方法的核心思想是，如果一个软件崩溃了，那么它就不应该继续运行，而是应该立即停止运行，然后由运维人员来修复它。
这种方法的优点是，它可以让软件开发人员更加专注于软件的核心功能，而不是在软件崩溃时花费大量的时间来修复它。
