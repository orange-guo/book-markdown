# Mapstruct

## Introduction

MapStruct是一个代码生成器，用于简化Java
bean之间的映射。它通过注解处理器生成映射代码，这些注解处理器在编译时生成代码，而不是在运行时生成代码。
这意味着MapStruct生成的代码不会增加应用程序的启动时间，也不会增加应用程序的运行时开销。
